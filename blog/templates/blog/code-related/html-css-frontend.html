HTML & CSS: Introduction

HTML and CSS go hand-in-hand as well as cereal and milk. While one (HTML) is essentially a markup language that is easy for humans to reproduce and comprehend, the other one (CSS) is a language for style sheets for describing a presentation of a certain document. CSS significantly improves the webdesign experience for both the programmer behind the page and the end user. This website uses CSS as well! Once you've got the hang of it, check out my website on Github to take a look at how CSS is used in the real world. Obviously, the HTML is there, too, however, it's not the easiest code to understand for a novice due to pythonic injections inside. You can always right-click and "View Page Source" though!

HTML

HTML is the foundation of web development, web design and, well, everything website-related. The language is extremely easy to grasp and is helpful to know even for non-tech people. But what is HTML? Over time, I've heard so many wrong explanations I decided to do something about it. Let's put it this way:

  • HTML stands for HyperText Markup Language.
  • HTML consists of a series of tags and elements which tell the browser how to display the content.
  • Tags separate page contents into distinctive pieces for the browser to display.
  • Instead of showing the tags, HTML uses them as an indication of how a certain element should look.
  • HTML is NOT a programming language.

Let's take a look at a very simple webpage and try to understand the structure. 

            
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>
