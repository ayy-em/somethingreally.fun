Adding digital clock to your website

To add a JS clock to your website, you need to solve three main issues.

1. Getting the current time.
2. Making it show up on your website.
3. Configuring it to properly update every second.

Assuming you guys have already read my Intro to JavaScript, you already know that the JS code should be put either in the 'script' tag or saved as a separate file with the '.js' extension. We then add the file to our HTML.
