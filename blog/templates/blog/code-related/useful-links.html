Useful Resources:
Guides, Communities and Tutorials

Learning to code sure is an uphill battle. Lucikly, the web is full of resources that would help you out in no time. Here's a list of resources you should love and cherish. I'll be updating this list as much as I can. Suggestions are always welcome!

{% load static %}

Github

Github is world's largest community of developers and a global company owned by Microsoft that provides hosting for software development version control using Git. You can host and review code, manage projects, and build software alongside 40 million developers. Github is great for version control, collaboration and showing off your crazy coding skills to potential employers kind of like a digital portfolio for designers.

Stack Overflow

Stack Overflow is the largest online community for anyone that codes to learn, share their knowledge, and build their careers. Visitors from all over the world come to Stack Overflow to help solve coding problems and develop new skills.
I'm pretty sure you'll encounter this website very early in your coding journey.

W3Schools

Another website you're bound to stumble upon when googling that pesky problem you're struggling with, W3Schools is an educational platform with a focus on web development. Content mostly includes tutorials about HTML, CSS, JavaScript and such.

CodePen

CodePen is an online community for testing and showcasing user-created HTML, CSS and JavaScript code snippets. It functions as an online code editor and open-source learning environment, where developers can create code snippets, called "pens," and test them. A very convenient way to showcase your beautiful new CSS background!
