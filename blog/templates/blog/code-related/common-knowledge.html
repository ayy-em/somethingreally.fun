Stuff Every Coder Must Know

{% load static %}

Frontend or backend, mobile or web, every developer, software engineer and, to an extent, data scientist, is expected to have a lot of "common" knowledge that's just expected of them. Think of it as the new "Microsoft Office"-like starter pack.

And even if you can, in theory, get by without them just fine, they still remain extremely valuable skills that are pretty easy to pick up. So, let's get started! Here's the full list:

  • Version Control (VCS) and Git: push, pull and merge
  • Basic HTML and CSS: a web page structure
  • Databases, Object-relational mapping (ORM) & SQL
  • Environmental variables
  • Bash, terminal, cmd
  • IDEs: VSC, iDEA, PyCharm and others
  • APIs: networking time!

I know this seems kind of overwhelming and leaves you with more questions than you previously had, but trust me - given enough time, dedication and patience, this will be a breeze!
